## 5. 对局状态机与规则实现

- [ ] 5.1 实现状态机：**无准备**；发言 → 投票（**8 秒**）→ 结算出局或全员弃票继续；平票加赛；**胜负判定**（卧底出局平民胜 / 剩 2 人含卧底则卧底胜）。
- [ ] 5.2 **投票**不可投自己；超时=弃票；**全员弃票**则无人出局、进入下一轮。
- [ ] 5.3 发言顺序：座位号升序；平票加赛仅平票玩家、顺序升序。
- [ ] 5.4 发言超时自动下一位或进入投票（时长可在配置中定义，与投票 8 秒区分）。
- [ ] 5.5 平票加赛无次数上限，直至可执行出局或非全员弃票的有效结果。
